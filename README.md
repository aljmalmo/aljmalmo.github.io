# موقع الجمال - النسخة المحسنة

## نظرة عامة

هذا هو الموقع الشخصي لمحمد جمال، مطور برمجيات متخصص في تطوير واجهات المستخدم. تم تطوير هذا الموقع باستخدام HTML5، CSS3، وJavaScript الحديث مع التركيز على التصميم المتجاوب وتجربة المستخدم المتميزة.

## الميزات الجديدة

### 🎨 التصميم والواجهة
- تصميم حديث ومتجاوب بالكامل
- نظام ألوان احترافي مع تدرجات جذابة
- رسوم متحركة ناعمة وتأثيرات بصرية
- دعم الوضع المظلم والفاتح
- خطوط عربية محسنة للقراءة

### 📱 تجربة المستخدم
- تصميم متجاوب يعمل على جميع الأجهزة
- قائمة تنقل ذكية للهواتف المحمولة
- أزرار تفاعلية مع تأثيرات hover
- تحميل سريع ومحسن للأداء
- إمكانية الوصول المحسنة (Accessibility)

### 🔧 الوظائف التقنية
- تكامل مع GitHub API لعرض المشاريع تلقائياً
- نظام بحث في المدونة
- تصنيف المقالات بالفئات
- مشاركة المحتوى على وسائل التواصل
- شريط تقدم القراءة في المقالات

### 📊 تحسين الأداء
- ضغط وتحسين الصور
- تحميل كسول للمحتوى
- استخدام CDN للمكتبات الخارجية
- تحسين SEO متقدم
- دعم PWA (Progressive Web App)

## الصفحات المتاحة

### 🏠 الصفحة الرئيسية (index.html)
- عرض المشاريع من GitHub تلقائياً
- إحصائيات المطور (عدد المشاريع، النجوم، اللغات)
- تصفية المشاريع حسب اللغة البرمجية
- روابط سريعة لوسائل التواصل

### 👤 من أنا (about.html)
- نبذة شخصية مفصلة
- المهارات والتخصصات
- الرحلة المهنية والخبرات
- معلومات التواصل

### 📝 المدونة (blog.html)
- قائمة المقالات مع معاينة
- نظام بحث في المحتوى
- تصنيف المقالات بالفئات
- مقال مميز في المقدمة

### 📄 سياسة الخصوصية (privacy-policy.html)
- سياسة شاملة وشفافة
- فهرس محتويات تفاعلي
- معلومات التواصل للاستفسارات

## كيفية الاستخدام

### النشر على GitHub Pages
1. ارفع الملفات إلى مستودع GitHub
2. فعل GitHub Pages من إعدادات المستودع
3. اختر المجلد الرئيسي كمصدر
4. الموقع سيكون متاحاً على `username.github.io`

### التخصيص
1. **الألوان**: عدل متغيرات CSS في بداية ملف `style.css`
2. **المحتوى**: حدث النصوص في ملفات HTML
3. **المشاريع**: سيتم جلبها تلقائياً من GitHub
4. **الروابط**: حدث روابط وسائل التواصل
