# موقع المحفظة الشخصية - دليل الاستخدام والتخصيص

## نظرة عامة

هذا موقع ويب احترافي أحادي الصفحة ومتجاوب باللغة العربية مصمم لعرض الخدمات الرقمية المتعددة. الموقع يتميز بتصميم حديث ونظيف مع تجربة مستخدم ممتازة على جميع الأجهزة.

## الميزات الرئيسية

- ✅ تصميم متجاوب (Responsive) يعمل على جميع الأجهزة
- ✅ دعم كامل للغة العربية مع إمكانية إضافة أقسام إنجليزية
- ✅ تحسين محركات البحث (SEO)
- ✅ حركات وتأثيرات سلسة
- ✅ نماذج اتصال تفاعلية
- ✅ أزرار تواصل ثابتة (واتساب وتليجرام)
- ✅ قسم أسئلة شائعة قابل للطي
- ✅ شرائح تمرير للشهادات
- ✅ تصميم قابل للتخصيص بسهولة

## التثبيت والإعداد

### 1. رفع الملفات

1. قم بتحميل جميع الملفات إلى خادم الويب الخاص بك
2. تأكد من الحفاظ على هيكل المجلدات كما هو
3. تأكد من أن جميع الصور في مجلد `assets/`

### 2. إعداد نموذج الاتصال

لتفعيل نموذج الاتصال، تحتاج إلى إعداد EmailJS:

1. سجل حساب مجاني في [EmailJS](https://www.emailjs.com/)
2. أنشئ خدمة بريد إلكتروني جديدة
3. أنشئ قالب رسالة
4. في ملف `script.js`، استبدل:
   ```javascript
   emailjs.init("YOUR_PUBLIC_KEY"); // استبدل بمفتاحك العام
   emailjs.send('YOUR_SERVICE_ID', 'YOUR_TEMPLATE_ID', templateParams)
   ```

### 3. تخصيص معلومات الاتصال

في ملف `index.html`، ابحث عن الأقسام التالية وقم بتحديثها:

```html
<!-- معلومات الاتصال -->
<p>your.email@example.com</p>
<p>+1234567890</p>
<p>@yourusername</p>

<!-- روابط التواصل الاجتماعي -->
<a href="https://wa.me/1234567890">
<a href="https://t.me/yourusername">
```

## التخصيص الأساسي

### تغيير الألوان

في ملف `styles.css`، يمكنك تعديل الألوان من خلال متغيرات CSS:

```css
:root {
  --primary-color: #1e3a8a;      /* الأزرق الداكن */
  --accent-color: #0d9488;       /* الأزرق المخضر */
  --accent-hover: #14b8a6;       /* لون التمرير */
  --text-color: #334155;         /* لون النص */
  --secondary-color: #f8fafc;    /* الخلفية الثانوية */
}
```

### تغيير الخطوط

```css
:root {
  --font-arabic: 'Cairo', 'Tajawal', sans-serif;
  --font-english: 'Inter', 'Roboto', sans-serif;
}
```

### تحديث المحتوى

#### 1. القسم الرئيسي (Hero Section)
```html
<h1 class="hero__title">
    مرحباً، أنا <span class="hero__title-accent">[اسمك]</span>
    <br>[وصف مختصر]
</h1>
```

#### 2. الخدمات
لإضافة خدمة جديدة، انسخ هيكل `service__card` وقم بتعديل:
- الأيقونة (`<i class="fas fa-icon-name"></i>`)
- العنوان والوصف
- السعر

#### 3. معرض الأعمال
لإضافة مشروع جديد، انسخ هيكل `portfolio__item` وقم بتحديث:
- الصورة
- العنوان والوصف
- الروابط

## الأداء والتحسين

### تحسين الصور
- استخدم تنسيق WebP للصور الحديثة
- ضغط الصور قبل الرفع
- استخدم أحجام مناسبة (الصورة الرئيسية: 800x600px، صور المعرض: 600x400px)

### تحسين محركات البحث
- قم بتحديث `meta description` و `keywords`
- أضف نصوص بديلة للصور
- استخدم عناوين HTML منظمة (H1, H2, H3)

## النشر

### 1. الاستضافة المجانية
- **GitHub Pages**: مجاني للمشاريع العامة
- **Netlify**: مجاني مع ميزات متقدمة
- **Vercel**: مجاني للمشاريع الشخصية

### 2. الاستضافة المدفوعة
- **SiteGround**: موصى به للمواقع العربية
- **Bluehost**: خيار اقتصادي
- **DigitalOcean**: للمطورين المتقدمين

### خطوات النشر على Netlify:
1. سجل حساب في [Netlify](https://netlify.com)
2. اسحب مجلد الموقع إلى لوحة التحكم
3. سيتم نشر الموقع تلقائياً
4. يمكنك ربط نطاق مخصص

## الصيانة والتحديث

### إضافة خدمات جديدة
1. أضف بطاقة خدمة جديدة في قسم الخدمات
2. أضف الخدمة إلى قائمة الاختيار في نموذج الاتصال
3. حدث دالة `getServiceName()` في JavaScript

### إضافة أعمال جديدة
1. أضف الصور إلى مجلد `assets/`
2. أضف بطاقة جديدة في قسم معرض الأعمال
3. تأكد من تحسين الصور

### تحديث الشهادات
1. أضف صور العملاء إلى مجلد `assets/`
2. أضف شريحة جديدة في قسم الشهادات
3. تأكد من استخدام أسماء وصور حقيقية (بإذن)

## الدعم الفني

### مشاكل شائعة وحلولها

#### 1. النموذج لا يرسل الرسائل
- تأكد من إعداد EmailJS بشكل صحيح
- تحقق من مفاتيح API
- تأكد من اتصال الإنترنت

#### 2. الصور لا تظهر
- تحقق من مسارات الصور
- تأكد من رفع مجلد `assets/`
- تحقق من أذونات الملفات

#### 3. التصميم لا يظهر بشكل صحيح
- تأكد من رفع ملف `styles.css`
- تحقق من وجود أخطاء في CSS
- امسح ذاكرة التخزين المؤقت للمتصفح

#### 4. JavaScript لا يعمل
- تحقق من وجود أخطاء في وحدة التحكم
- تأكد من رفع ملف `script.js`
- تحقق من تحميل المكتبات الخارجية

### أدوات مفيدة للتطوير
- **VS Code**: محرر نصوص متقدم
- **Chrome DevTools**: لفحص وتصحيح الأخطاء
- **GTmetrix**: لقياس سرعة الموقع
- **Google PageSpeed Insights**: لتحسين الأداء

## الأمان

### نصائح أمنية مهمة
- لا تشارك مفاتيح API في الكود العام
- استخدم HTTPS دائماً
- قم بتحديث المكتبات بانتظام
- استخدم نماذج محمية ضد SPAM
