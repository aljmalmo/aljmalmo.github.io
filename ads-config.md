# دليل استخدام نظام الإعلانات

## نظرة عامة

تم تطوير نظام إعلانات متقدم وسهل الاستخدام للموقع، يتيح لك إضافة وإدارة الإعلانات في مواقع مختلفة بسهولة.

## أماكن الإعلانات المتاحة

### 1. إعلان تحت الهيدر (Header Ad)
- **الموقع**: يظهر مباشرة تحت شريط التنقل الرئيسي
- **المعرف**: `headerAd`
- **الاستخدام**: مثالي للإعلانات البارزة والعروض المهمة

### 2. إعلان فوق الفوتر (Footer Ad)
- **الموقع**: يظهر قبل الفوتر مباشرة
- **المعرف**: `footerAd`
- **الاستخدام**: مناسب للإعلانات الثانوية أو الروابط التابعة

### 3. إعلان أعلى المقال (Article Top Ad)
- **الموقع**: يظهر في بداية المقال قبل المحتوى
- **المعرف**: `articleTopAd`
- **الاستخدام**: إعلانات ذات صلة بموضوع المقال

### 4. إعلان وسط المقال (Article Middle Ad)
- **الموقع**: يظهر في منتصف المقال تقريباً
- **المعرف**: `articleMiddleAd`
- **الاستخدام**: إعلانات تفاعلية أو محتوى مدعوم

### 5. إعلان أسفل المقال (Article Bottom Ad)
- **الموقع**: يظهر في نهاية المقال
- **المعرف**: `articleBottomAd`
- **الاستخدام**: إعلانات المقالات ذات الصلة أو الاشتراكات

## كيفية إضافة نظام الإعلانات للصفحات

### الخطوة 1: إضافة ملف JavaScript
أضف هذا السطر قبل إغلاق تاج `</body>` في كل صفحة:

```html
<script src="ads.js"></script>
```

### الخطوة 2: التحقق من التفعيل
النظام يعمل تلقائياً عند تحميل الصفحة. يمكنك التحكم في تفعيل/إلغاء تفعيل الإعلانات من خلال ملف `ads.js`.

## كيفية تخصيص الإعلانات

### تعديل كود الإعلان

1. افتح ملف `ads.js`
2. ابحث عن القسم المطلوب في `AdsConfig.ads`
3. استبدل المحتوى داخل `<!-- ضع كود الإعلان هنا -->`

**مثال لإضافة إعلان Google AdSense:**

```javascript
headerAd: {
    enabled: true,
    code: `
        <div class="ad-container header-ad">
            <div class="ad-label">إعلان</div>
            <div class="ad-content">
                <script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
                <ins class="adsbygoogle"
                     style="display:block"
                     data-ad-client="ca-pub-xxxxxxxxxx"
                     data-ad-slot="xxxxxxxxxx"
                     data-ad-format="auto"></ins>
                <script>
                     (adsbygoogle = window.adsbygoogle || []).push({});
                </script>
            </div>
        </div>
    `
}
```

### تفعيل/إلغاء تفعيل إعلان معين

```javascript
// لإلغاء تفعيل إعلان الهيدر
AdsConfig.ads.headerAd.enabled = false;

// أو استخدام الدالة المساعدة
toggleAd('headerAd', false);
```

### إلغاء تفعيل جميع الإعلانات

```javascript
// في ملف ads.js
AdsConfig.enabled = false;

// أو استخدام الدالة المساعدة
removeAllAds();
```

## الدوال المساعدة المتاحة

### `updateAd(adType, newCode)`
تحديث كود إعلان معين

```javascript
updateAd('headerAd', '<div>كود الإعلان الجديد</div>');
```

### `toggleAd(adType, enabled)`
تفعيل أو إلغاء تفعيل إعلان معين

```javascript
toggleAd('footerAd', false); // إلغاء تفعيل
toggleAd('footerAd', true);  // تفعيل
```

### `removeAllAds()`
إزالة جميع الإعلانات من الصفحة

```javascript
removeAllAds();
```

## التصميم المتجاوب

النظام يدعم التصميم المتجاوب تلقائياً:
- على الشاشات الكبيرة: إعلانات بحجم كامل
- على الهواتف المحمولة: إعلانات مُحسَّنة للشاشات الصغيرة

## أفضل الممارسات

### 1. التوازن في عدد الإعلانات
- لا تفرط في عدد الإعلانات
- اترك مساحة كافية بين الإعلانات والمحتوى

### 2. الصلة بالمحتوى
- اختر إعلانات ذات صلة بموضوع الموقع
- استخدم إعلانات تهم جمهورك المستهدف

### 3. تجربة المستخدم
- تأكد من أن الإعلانات لا تعيق تجربة القراءة
- استخدم أحجام مناسبة للشاشات المختلفة

### 4. الأداء
- تجنب الإعلانات الثقيلة التي تبطئ الموقع
- استخدم تقنيات التحميل الكسول عند الحاجة

## استكشاف الأخطاء

### الإعلانات لا تظهر
1. تأكد من أن `AdsConfig.enabled = true`
2. تحقق من أن الإعلان المحدد مفعل
3. تأكد من صحة كود HTML للإعلان

### مشاكل التصميم
1. تحقق من أنماط CSS المخصصة
2. تأكد من عدم تعارض الأنماط مع تصميم الموقع
3. اختبر على أجهزة مختلفة

### مشاكل الأداء
1. قلل من عدد الإعلانات في الصفحة الواحدة
2. استخدم إعلانات محسنة للسرعة
3. فعل ضغط الملفات

## الدعم والتطوير

لأي استفسارات أو مشاكل:
1. راجع هذا الدليل أولاً
2. تحقق من وحدة تحكم المتصفح للأخطاء
3. اختبر على متصفحات مختلفة

## مثال كامل للتطبيق

```html
<!DOCTYPE html>
<html>
<head>
    <title>صفحة مع إعلانات</title>
</head>
<body>
    <!-- محتوى الصفحة -->
    
    <!-- إضافة نظام الإعلانات -->
    <script src="ads.js"></script>
    
    <!-- تخصيص إضافي (اختياري) -->
    <script>
        // تخصيص إعلان معين
        updateAd('headerAd', '<div>إعلان مخصص</div>');
        
        // إلغاء تفعيل إعلان معين
        toggleAd('footerAd', false);
    </script>
</body>
</html>
```
